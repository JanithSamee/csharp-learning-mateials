# Control Structure

### For loop

```csharp
for (int i = 0; i < 16; i++){
    Console.WriteLine("Number Count: "+ i);
}
```

### while loop

```csharp
int i = 20;
while ( i>0){
    Console.WriteLine("Number Count: "+ i);
    i = i - 1; //Or i--
}
```

### if-else

```csharp
if(condition){
    Console.WriteLine("True Statement");
}
else{
    Console.WriteLine("False Statement");
    
}
```
