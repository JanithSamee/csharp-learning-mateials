# Variables

### Declare a variable 
Variable is defined as `type variable_name = value` .\
```csharp
string name = "Peter"
```
### Variable types
```csharp
int year = 2022;
long weight = 1000000000000L;
float spead =60.25F;
double discount = 7.89D;
char firstChar = 'J';
bool isWritten = true;
string color = "Yellow";
```

* int - 4 bytes
* long - 8 bytess
* float - 4 bytes
* bool - 1 bit
* char - 2 bytes
* string -2 bytes per character
